PARAMETRIZAÇÃO DE DISPOSITIVOS DE PROTEÇÃO EM PLATAFORMAS DE PETRÓLEO (13,8 KV)?? Escopo do DocumentoEste documento apresenta as recomendações técnicas para parametrização de dispositivos de proteção em sistemas elétricos de potência de 13,8 kV, com aplicação direta em plataformas de petróleo offshore. Inclui referências normativas nacionais e internacionais, com foco em seletividade, sensibilidade e integração com sistemas inteligentes.?? Considerações Finais* O sistema deve garantir seletividade, sensibilidade e redundância.* A atuação dos dispositivos deve estar coordenada com a lógica do RL (Aprendizado por Reforço).* Recomenda-se registrar todos os parâmetros de atuação em arquivos .json para controle do RL.* A interface frontend deve permitir ajuste em tempo real, dentro dos limites normativos.?? Manual de Integração Proteção + IA (RL)1. Entradas do Agenteo Estado das zonas (Z1, Z2)o Correntes medidas (CTs)o Estado dos relés (ligado, atuado, bloqueado)o Estado dos disjuntores (aberto, fechado)2. Ações Possíveiso Ajustar pickup / TMS dos relés 51o Alterar curva (NI, VI, EI)o Reprogramar tempo dos disjuntoreso Reconfigurar conexão zona-carga3. Função de Recompensao +10: Atuação seletiva correta (87T ou 51 conforme falha)o +5: Backup atuando após falha de primárioo -20: Carga crítica desligada indevidamenteo -50: Blackout geral na rede4. Formato de Integraçãoo Todos os parâmetros em .jsono APIs de leitura/escrita com FastAPIo Interface frontend com botões de inspeção e ajuste manualAutor: Angelo C.
